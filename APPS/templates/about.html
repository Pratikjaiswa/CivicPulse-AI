About the Project

Smart Civic Complaint Intelligence Platform

CivicPulse AI is an end-to-end web application that uses Machine Learning to classify civic complaints, store them in MySQL, assign departments and officers, track complaint progress, and generate analytics.

AI Classification

Predicts complaint category automatically from user-submitted text.

Complaint Tracking

Allows users to track complaint status, officer assignment, and admin remarks.

Admin Workflow

Admins can manage priority, status, department, officer, and resolution updates.

Analytics Dashboard

Visualizes complaint trends, status distribution, workload, and key insights.

Technology Stack

Developed using Python, Flask, MySQL, Machine Learning and Chart.js.
